# VolunteerHub - Event Volunteer Management System

## 📌 Overview

VolunteerHub is a **3‑Tier Event Volunteer Management Web Application** that connects **Event Organizers** and **Volunteers**. Organizers can create and manage events, while volunteers can browse and register for participation.

---

## 🏗️ Architecture

* **Frontend:** HTML, CSS, JavaScript (AWS S3)
* **Backend:** Spring Boot REST API (AWS EC2)
* **Database:** MySQL (AWS RDS / MySQL)

---

## ✨ Features

### Admin

* Create & Manage Events
* View Volunteers
* Manage Registrations

### Volunteer

* Register & Login
* View Events
* Apply for Events

---

## 🛠️ Tech Stack

* HTML, CSS, JavaScript
* Spring Boot (Java)
* MySQL
* AWS (S3, EC2, RDS)
